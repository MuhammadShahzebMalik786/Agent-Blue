# Agent Blue 🤖🌐

**Automates the browser. You don't lift a finger.**

Agent Blue is a powerful, intelligent browser extension that acts as your personal web automation assistant. By leveraging state-of-the-art AI models, Agent Blue completely automates your browser. Just tell it what you want to do in plain English, and it will navigate, click, scroll, and type for you!

## ✨ Features
* **Natural Language Automation:** Simply type your goal (e.g., "Find the cheapest flights to New York and save the top 3 options to memory").
* **Multi-Model Support:** Seamlessly switch between different AI brains based on your needs:
  * **Google Gemini:** Fast, powerful, and free-tier friendly.
  * **GPT-4o / GitHub Models:** Advanced reasoning capabilities.
  * **Groq (Llama/Mixtral):** Lightning-fast inference.
  * **Ollama (Local):** 100% private, local automation using models running directly on your machine.
* **Smart Page Understanding:** Analyzes the active DOM to figure out exactly what buttons to click and what fields to type in.
* **Memory Scratchpad:** Capable of extracting text from pages and saving it to a local memory bank to use across different tabs and websites.
* **Bring Your Own Key (BYOK):** Complete control over your usage. Plug in your own API keys for your preferred AI provider to ensure cost-effectiveness and privacy.

## 🚀 Installation (Unpacked Extension)
Since this extension interacts with powerful APIs and executes scripts, you can run it locally in developer mode:

1. Clone or download this repository.
2. Open your Chromium-based browser (Chrome, Edge, Brave).
3. Navigate to the extensions page:
   * Chrome: `chrome://extensions/`
   * Edge: `edge://extensions/`
4. Enable **Developer mode** (usually a toggle in the top right).
5. Click **Load unpacked** and select the folder containing this extension (`Ai Agent Blue`).
6. Pin the Agent Blue icon to your toolbar for easy access!

## ⚙️ Usage
1. Open the **Agent Blue** popup.
2. Select your preferred **AI Provider** (Gemini, Groq, GitHub Models, or Ollama).
3. Enter your **API Key** (If using Ollama, ensure your local server is running and leave the Host URL default).
4. Click **Refresh** next to the AI Model dropdown to fetch the latest available models.
5. Type your instruction into the "What should I do?" box.
6. Click **Start Agent** and watch it work!

## 🔒 Privacy & Security
Agent Blue runs **entirely on your local machine**. 
* We do not collect, store, or sell any of your personal data.
* API keys and configuration settings are stored securely in your browser's local storage.
