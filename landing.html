AI Agent Blue is a Chrome extension that uses AI to navigate, click, type, and remember — fully automating any website for you. No code required.

🐙 View on GitHub ✉️ Contact Developer

What it does

Everything Your Browser Can Do — Automated

🌐

Smart Navigation

AI navigates to any URL, handles redirects, and waits for pages to load — just like a human.

🖱️

Click & Type

Clicks buttons, fills forms, and types text into any field — even React and Vue apps.

🧠

Memory Across Sites

Remembers data from one website and uses it on another — extract, store, reuse.

🧑‍💻

Human Node

Detects CAPTCHAs and pauses for you to solve them, then resumes automatically.

⚡

Multi-Provider AI

Switch between Google Gemini, GitHub GPT-4o, and Groq — use whichever has free quota.

🔄

Auto Fallback

Hits a rate limit? Automatically switches to the next available model and keeps going.

Supported AI Engines

Pick Your AI. We Handle the Rest.

🔵 Google Gemini ⚫ GitHub GPT-4o 🟠 Groq (Llama / Mixtral)
